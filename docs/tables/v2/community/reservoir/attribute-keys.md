# 属性键

## **reservoir.属性键**

该表包含有每个属性键的记录

查询的示例可以在这里找到:

[https://dune.com/queries/1302930/2232305](https://dune.com/queries/1302930/2232305)

| **列名** | **类型**  | **说明**                          |
|-----------------|-----------|------------------------------------------|
| id              | 字符串    | 内部属性键ID                |
| collection\_id  | 字符串    | 相关集合ID                 |
| key             | 字符串    | 键名称                |
| kind            | 字符串    | 值类型 (字符串, 数字, 日期, 范围) |
| rank            | 字符串    | 排序顺序                               |
| created\_at     | 时间戳 | 创建属性键的时间戳  |
| updated\_at     | 时间戳 | 更新属性键的时间戳  |
